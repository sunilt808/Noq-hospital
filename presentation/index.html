ALGORITHM

Atomic Queues

To prevent catastrophic race conditions when multiple patients book simultaneously, NOQ bypasses standard reads and utilizes MongoDB's native atomic operators.

  • Guarantees zero duplicate tokens
  • Instantly updates waiting count
queue_service.py 
await db.queues.find_one_and_update(
    {"_id": queue_id},
    {"$inc": {
        "token_counter": 1,
        "waiting_count": 1
    }},
    return_document=ReturnDocument.AFTER
)
